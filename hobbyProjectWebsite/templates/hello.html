Linear Motion Cabinet

Hobby project to build a cabinet that has uses a Raspberry Pi and a worm gear to raise and lower the cabinet. The ultimate goal being to be able put a projector in the cabinet and be able to hide the projector when it is not in use. I am planning on incorporating commands into a Google Home to be able to send request to the Raspberry Pi.
