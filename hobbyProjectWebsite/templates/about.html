About Me

Anthony Jacques

Hi there! This website is meant to show some of my programming skills in addition to some of my favorite projects that I've worked on outside of my job! I like to tinker with different things from Python to wood working!

I primarily use Python and structured text. I also use C#, SQL, Jython, and ladder logic at my job. In my hobby projects, I have used/am using C, Arduino, Assembly, and G-Code. I have experience with object oriented programming since working in the physical world lends itself nicely to creating objects and classes.

While the programming languages might be different in industrial automation, the thought processes, critical thinking, and software practices are very similar to higher level languages. As an example, I led the development of a factory-wide tracking system built on Python and MySQL. I also drove the automating of our workflow to eliminate tedious and error-prone processes using Python and C#.
